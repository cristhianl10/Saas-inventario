# Assets - Imágenes de Sublirium

## Logo de la App

Coloca aquí el logo circular de Sublirium con el nombre:
- `sublirium_logo.png` (recomendado: 512x512px o mayor, formato PNG con fondo transparente)

El logo se mostrará de forma circular en la esquina superior derecha del AppBar.

## Formato recomendado
- Tamaño: 512x512px o mayor (se escalará automáticamente)
- Formato: PNG con transparencia
- El logo debe ser cuadrado, la app lo recortará en forma circular

Si no se encuentra el logo, la app mostrará una "S" con el gradiente de Sublirium como fallback.
